# SPOT

SPOT — современное веб/PWA-приложение для сохранения, организации и визуализации пользовательских мест на интерактивной карте. Пользователи могут отмечать посещенные и запланированные локации, делиться ими с друзьями, создавать совместную карту пары и получать персонализированную ленту активности.

## Основные возможности

- Авторизация и регистрация пользователей
- Интерактивная карта с пользовательскими метками
- Создание, редактирование и удаление мест
- Загрузка фотографий и выставление оценок
- Статусы посещения («Посетил» / «Планирует посетить»)
- Совместная карта пары
- Подписки на пользователей и пары
- Персонализированная лента активности
- Публичные и приватные профили
- Адаптация под мобильные устройства и поддержка PWA
- Административная панель для управления системой

## Архитектура проекта

Проект состоит из клиентской и серверной частей.

### Frontend

Клиентская часть отвечает за отображение пользовательского интерфейса, взаимодействие с картой и работу с REST API.

**Стек технологий**

- React
- TypeScript
- Vite
- Redux Toolkit Query
- Mantine UI
- React Router
- MapLibre + react-map-gl
- PWA (Service Worker, Manifest)

### Backend

Серверная часть предоставляет REST API, реализует бизнес-логику приложения, авторизацию пользователей, работу с картой, загрузку изображений и хранение данных.

**Стек технологий**

- NestJS
- Prisma ORM
- PostgreSQL
- JWT Authentication
- Docker
- Nginx

## Инфраструктура

- VPS-сервер
- Docker Compose
- Nginx
- HTTPS
- GitHub Actions
- CI/CD

## Назначение проекта

SPOT предназначен для сохранения личных и совместных воспоминаний о местах, поиска новых интересных локаций через друзей и организации собственной интерактивной карты посещенных и запланированных мест.
